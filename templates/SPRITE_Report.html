● Methods

Analysis pipeline description and software versions

Pipeline Overview

The SPRITE (Split-Pool Recognition of Interactions by Tag Extension) pipeline processes paired-end sequencing reads to identify multi-way chromatin contacts. The workflow proceeds as follows:

  1. Adapter trimming with Trim Galore
  2. Barcode identification with BarcodeIdentification v1.2.0
  3. Full-barcode filtering and DPM trimming with Cutadapt
  4. DNA alignment with Bowtie2 (MAPQ ≥ 20)
  5. Chromosome annotation conversion (Ensembl → UCSC)
  6. Repeat-element masking with bedtools
  7. Cluster assembly with get_clusters.py
  8. Contact matrix construction with get_sprite_contacts.py
  9. ICE normalisation with HiCorrector 1.2
  10. Heatmap plotting with R / ggplot2
